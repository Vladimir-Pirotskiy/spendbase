# React + TypeScript + Vite + ReduxToolKit + Vitest + TailwindCSS

### Description:

in project implemented:

- React for UI
- Redux Toolkit for state managment
- Redux-persist to save and fetch the state
- Vitest for test
- Vite
- Vercel
- Tailwind, Antd, Radix-Ui

### run project

>
> `yarn run dev`
>

### run test

> `yarn run test`
>
